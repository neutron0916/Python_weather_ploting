# Weather chart app

## Purpose 
get the temp and PoP data from 中央氣象資料開放平台 and plot it


## python requirement 
* eel --> GUI tool
* pandas --> handle the data
* requests --> get data
* json --> load the data


## CDN requirement: 
* plotly --> "https://cdn.plot.ly/plotly-latest.min.js"
* jQuery --> "https://code.jquery.com/jquery-3.5.1.min.js"
